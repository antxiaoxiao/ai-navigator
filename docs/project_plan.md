## 商业计划（v0 验证后再讨论）

**触发条件：** v0 建完 + 自己用两周 + 有第一批真实反馈

**到时候要回答的问题：**
1. 你自己每天都想打开它吗？
2. 身边的人看到后有什么反应？
3. 什么功能真的有用，什么没人碰？

**初步倾向的方向：**
先个人后企业——从个人用户开始验证，
积累口碑和数据后再考虑企业采购。

**现在需要做的一件事：**
查一下你的劳动合同，确认有没有知识产权相关条款。
在确认之前，用自己的设备、自己的时间、自己的账号开发。
